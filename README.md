# SkillSwap — AI-Powered Skill Exchange Platform

A backend system for peer-to-peer skill exchange with voice-based interaction.

## Tech Stack
Python · Django REST Framework · SQLite · JWT · Ultravox Voice AI

## Features
- 10+ REST APIs for user registration, skill management and booking
- JWT-based authentication with role-based access control
- Ultravox Voice AI integration for hands-free skill search
- Normalized database schema with optimized query handling

## Setup
cd swapskill
pip install -r requirements.txt
python manage.py runserver
